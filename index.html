Paper Design

About

About: This project was designed to show-case my design process, which turned out to show that I rely on much of my previous design skills. In this case, my illustration skills are prevelant.I believe that the best way to captivate someone’s eye is through white space because it allows the user to focus on the information at hand, also creating a very valuable user experience.

Objective + proccess

Objective: Create paper styled user Interfaces
Process: conceptualizing I brainstormed ideas, which might benefit my portfolio, leading towards data visualization. ResearchFor this project I used Human Interface Guideline and wrote down many notes. Ideas I wrote many thought and then kept the best in order to solve a problem I set for myself, which was to cross analyze 2 companies information.flow I then wrote down the flow I would be designing, which I cam back to before finalizing for design.Style Tile + other things I then create a sort of brand to go by, which I kept throughout the design and kept to the same color scheme and text size indicated from HIG. I read about Lightning Design System and worked with that in mind as well. Sketching + grid Sketching was done after writing down the flow and the grid was created with a sketch, somewhat and then finalized in illustrator.Designing I wanted the design to have much white space and a great user experience, so I mad the design clean and easy to follow. Prototyping This is a step I am currently on.
